You can install the package via composer:

```bash
composer require rappasoft/lockout
```

## Version Compatibility

| Laravel | Lockout |
|:--------|:--------|
| 6.x     | 1.x     |
| 7.x     | 2.x     |
| 8.x     | 3.x     |
| 9.x     | 4.x     |
| 10.x    | 5.x     |
